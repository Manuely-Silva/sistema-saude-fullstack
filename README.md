# Sistema Saúde Fullstack

Sistema web para gerenciamento de pacientes e consultas médicas, desenvolvido com foco em aprendizado de desenvolvimento fullstack.

---

## Funcionalidades

- Cadastro de pacientes
- Listagem de pacientes
- API REST com Flask
- Banco de dados SQLite
- Integração frontend + backend
- Estrutura preparada para login e agendamentos

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Python (Flask)
- SQLite
- Flask-CORS
